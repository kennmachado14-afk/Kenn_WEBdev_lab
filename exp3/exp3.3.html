Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented, and functional programming. Python is often described as a "batteries included" language due to its comprehensive standard library.

print("Hello, World!")
Getting Started

To begin programming in Python, you first need to install Python on your computer. You can download the latest version from the official Python website at python.org.

Once installed, you can run Python in two main ways: using the interactive interpreter or by running Python scripts. The interactive interpreter allows you to test code snippets immediately, while scripts let you save and execute longer programs.

To verify your Python installation, open a terminal or command prompt and type python --version or python3 --version. This will display the installed Python version.

  • Download Python from the official website
  • Install Python following the installation wizard
  • Verify installation using the command line
  • Choose an integrated development environment (IDE) like PyCharm, VS Code, or use a simple text editor
Variables and Data Types

In Python, variables are created when you assign a value to them. Unlike some other programming languages, you don't need to declare the type of a variable explicitly. Python automatically determines the type based on the value assigned.

Python supports several built-in data types including integers, floats, strings, booleans, lists, tuples, sets, and dictionaries. Each data type has its own characteristics and use cases.

# Integer
age = 25

# Float
price = 19.99

# String
name = "Alice"

# Boolean
is_active = True
  • Integers: whole numbers like 42, -7, 0
  • Floats: decimal numbers like 3.14, -0.5
  • Strings: text enclosed in quotes
Control Flow

Control flow statements allow you to control the execution of your program based on conditions. Python provides several control flow tools including if statements, for loops, and while loops.

The if statement is used to execute code conditionally. You can combine it with elif (else if) and else clauses to handle multiple conditions. Python uses indentation to define code blocks, which makes the code highly readable.

# If statement
if temperature > 30:
    print("It's hot outside")
elif temperature > 20:
    print("Nice weather")
else:
    print("It's cold")

Loops are used to repeat a block of code multiple times. The for loop is commonly used to iterate over sequences like lists, tuples, or strings. The while loop continues executing as long as a condition remains true.

# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1
Functions

Functions are reusable blocks of code that perform specific tasks. In Python, you define a function using the def keyword, followed by the function name and parentheses containing any parameters.

Functions can accept parameters (inputs) and return values (outputs). They help organize code, make it more modular, and reduce repetition. Python also supports default parameter values and keyword arguments.

def greet(name, greeting="Hello"):
    """This function greets a person"""
    return f"{greeting}, {name}!"

# Call the function
message = greet("Alice")
print(message) # Output: Hello, Alice!
  • Use descriptive function names that indicate what the function does
  • Include docstrings to document your functions

Python functions are first-class objects, meaning they can be assigned to variables, passed as arguments to other functions, and returned from functions. This enables powerful programming patterns like decorators and higher-order functions.

Object Oriented Programming

Object-oriented programming (OOP) is a programming paradigm that organizes code around objects and classes. A class is a blueprint for creating objects, while an object is an instance of a class containing data and methods.

Python supports OOP concepts including encapsulation, inheritance, and polymorphism. Classes allow you to bundle data and functionality together, making your code more organized and maintainable.

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        return f"{self.name} says Woof!"

# Create an object
my_dog = Dog("Buddy", 3)
print(my_dog.bark())

The __init__ method is a special method called a constructor that initializes new objects. The self parameter refers to the instance of the class and is used to access attributes and methods within the class.

Inheritance allows you to create new classes based on existing classes, inheriting their attributes and methods. This promotes code reuse and establishes relationships between classes.
